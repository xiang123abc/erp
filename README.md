# erp

### 介绍
ERP网上平台

### 软件架构
* 核心框架：SpringBoot 2.0.0
* 持久层框架：Mybatis 1.3.2
* 日志管理：Log4j 2.10.0
* JS框架：Jquery 1.8.0
* UI框架: EasyUI 1.3.5

* 项目管理框架: Maven 3.2.3

### 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: Eclipse
* DB: Mysql5.7.4
* JDK: JDK1.8
* Maven: Maven3.2.3+

### 运行环境
* 数据库服务器：Mysql5.7.4
* JAVA平台: JRE1.8
* 操作系统：Windows、Linux等
